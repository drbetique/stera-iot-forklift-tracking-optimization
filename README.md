\# Stera IoT Forklift Tracking System



\## Project Overview

Real-time IoT solution for forklift tracking and logistics optimization using ESP32 microcontroller and cloud-based analytics.



\## Team

\- \*\*Scrum Master \& Lead Developer:\*\* \[Victor Betiku]

\- \*\*Developers:\*\* 4 team members

\- \*\*Duration:\*\* 12 weeks (6 sprints)



\## Technology Stack



\### Hardware

* ESP32 DevKit (ESP32-WROOM-32)
* Qorvo DWM1001-DEV UWB Boards
* MPU6050 Accelerometer/Gyroscope
* RC522 RFID Reader \& RFID Key card
* HC-SR04 Ultrasonic Sensors (4×)
* MicroSD card Module



\### Backend

\- Node.js + Express.js

\- MongoDB Atlas (database)

\- InfluxDB (time-series data)

\- MQTT Protocol



\### Frontend

\- React 18 (Web Dashboard)

\- React Native (Mobile App)

\- Leaflet.js (Maps)

\- Chart.js (Analytics)



\### Cloud Services

\- MongoDB Atlas

\- Railway.app (Backend hosting)

\- Vercel (Frontend hosting)



\## Project Structure

```

stera-iot-project/

├── frontend/         # React web dashboard

├── mobile/           # React Native mobile app

├── hardware/         # ESP32 firmware

└── docs/             # Documentation

```



\## Features

\- Real-time GPS tracking

\- Intelligent activity detection (Parked, Idle, Driving, Working)

\- RFID station identification

\- Fork height monitoring

\- Load detection

\- Collision avoidance

\- Analytics dashboard

\- Historical data \& reports
